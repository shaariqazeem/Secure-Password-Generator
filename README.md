# Secure Password Generator (Java)

A Java-based password generator that creates strong, randomized passwords using customizable character sets including uppercase letters, numbers, and special symbols.

## Features
- Generates secure random passwords
- Custom character set (letters, numbers, symbols)
- User input using Scanner
- Input validation to prevent errors

## Tech Stack
- Java
- java.util.Random
- Scanner

## How to Run
1. Compile:
   javac PasswordGenerator.java
2. Run:
   java PasswordGenerator
